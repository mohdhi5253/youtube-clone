🎬 YouTube Clone (HTML & CSS Project)

📌 Project Description

This is a **YouTube Clone UI project** built using **HTML and CSS**.
It replicates the layout of YouTube including header, sidebar, and video grid.

---

🚀 Live Demo

👉http://127.0.0.1:5501/index.html

---

 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox & Grid Layout

---

 ✨ Features

* Responsive YouTube-like layout
* Header with search bar & icons
* Sidebar navigation
* Video grid with thumbnails
* Clean and modern UI

1. Download or clone the repository
